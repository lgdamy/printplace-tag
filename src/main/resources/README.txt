Produto desenvolvido por Damy Tecnologia da Informação LTDA

Windows: Utilizar o arquivo .exe (Caso você não tenha uma jre, será direcionado para o link de download)
Multiplataforma: executar o .jar com o java (Necessita de no mínimo uma jre8) -> java -jar SkyPrint.jar

Código aberto, não licenciado.

Funcionamento:

1) Escolha a folha utilizada para impressão: A4, A3 (...)

2) Escolha as medidas (mm) do material imprimível (Altura x Largura).

3) Escolha o comprimento da margem e do espaçamento entre o material.

4) Calcular...

5) Os botões à direita servem para alterar a orientação da página, alterar o formato do material,
e alterar o tema da página de modelo.

6) O ícone pequeno à direita (verde ou vermelho) indica se a orientação desejada vai prover o máximo de material
de acordo com a orientação da página escolhida.
